# Solvere

The marketing site for **Solvere** — AI-native healthcare claim recovery for UAE clinics.

Built with Next.js 14 (App Router), Tailwind CSS, Framer Motion, and Inter + Fraunces.

## Develop

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Build

```bash
npm run build
```

## Deploy

Configured for Netlify via `netlify.toml` (uses `@netlify/plugin-nextjs`).
